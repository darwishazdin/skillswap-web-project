# SkillSwap - Web Development Project

## Project Overview
SkillSwap is a community-driven web platform that enables users to exchange skills without monetary transactions. Built as part of our Web Application Development I course at The American University of Kurdistan.

## Team Culers
- Darwish Khalil - Homepage, About Page, Navigation System, Form Validation
- Intisar Farman - Browse Skills, Profile Pages, Responsive Design, Mobile Menu
- Murad Saleh - Exchange System, Authentication, JavaScript Features, Local Storage

## Technologies Used
- HTML5 - Semantic Structure
- CSS3 - Responsive Design & Styling
- JavaScript - Interactive Features
- Local Storage - Data Persistence

## Project Structure
- index.html (Homepage)
- login.html (Authentication) 
- browse-skills.html (Skill Browser)
- profile.html (User Profiles)
- exchange.html (Exchange System)
- about.html (About & Contact)
- css/style.css (Main Styles)
- css/responsive.css (Mobile Design)
- js/main.js (Common Functions)
- js/browse.js (Skill Filtering)
- js/exchange.js (Form Handling)
- assets/ (Media Files)

## Features Implemented
- 6 Responsive Pages
- JavaScript Interactivity (Search, Filters, Forms)
- User Authentication Simulation
- Skill Exchange System
- Mobile-First Design
- Form Validation
